# Kalenderin

Kalender Indonesia modern berbasis web.

## Features

* Kalender bulanan
* Hari libur nasional Indonesia
* Kalender Hijriyah
* Weton Jawa
* Responsive mobile
* PWA installable

## Tech Stack

* ReactJS
* Vite
* TailwindCSS
* date-fns

## Development

```bash
npm install
npm run dev
```

## Branch Strategy

* `main` → production
* `development` → active development
